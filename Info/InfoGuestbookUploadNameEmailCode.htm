Gästebuch: Name, E-Mail und Code
Pflichtfelder für das Hochladen eines Bildes sind Name(n) und eine E-Mail-Adresse.

Die E-Mail-Adresse wird für das Senden vom Code verwendet. Die Adresse wird im Gästebuch nicht gezeigt.

Wenn Sie mit dem Text und/oder Bild nicht zufrieden sind, können Sie den letzten Beitrag selbst ändern.

Eine Änderung ein bestehendes Beitrag kann vom Gästebuch Administrator verlangt werden.

Falls Sie noch Hilfe brauchen, zeigen wir gerne an einem Konzert wie diese App funktioniert.

Es gibt auch die Möglichkeit einen Beitrag (Bild, Titel, Text) für das Gästebuch an uns zu senden. E-Mail-Adresse: guestbook@jazzliveaarau.ch.

Ablauf
1. Name(n) und E-Mail-Adresse eingeben.

2. 'Code verlangen' klicken.


3. E-Mail Programm öffnen und Code lesen
Betreff: JAZZ live AARAU Gaestebuch Code 79213

4. Code eingeben und 'Weiter' klicken
                  oder
   'Änderung eines bestehenden Beitrags' klicken
