The following things have been changed in the new version:

- There is an information file for each 'page'
- There is a title for each of the four pages
- The forward button for page 2 (upload of image) is displayed when the image has been loaded
- After upload of a record the user gets a message saying that there is function to change the last record
- The option to edit the lasr record has been added


After selecting the edit function and klicking of the open button 'page two' (select image) opens.
The user may however go back to 'page one' and change the name(s).
Please note that the name(s) of the last record is displayed.
It is not necessary for the user to use the same name(s) when requesting the code for editing the last record.

It the email address is changed on 'page one' after getting the code a new code must be requested.
For such an email address change the user gets a warning and the possibility to cancel the change
